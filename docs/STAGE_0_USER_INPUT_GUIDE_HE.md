# מה צריך ממך לסגירת שלב 0

סיפקת כבר את קבצי האודיו. זה מצוין.

כדי שאוכל לבצע בדיקת איכות אמיתית למודל הראשי, חסר רק `Ground Truth` (תמליל אמת).

## מה להכין עכשיו (מינימום)

1. לפתוח את הקובץ: `data/bakeoff/annotation_batch_30.csv`
2. עבור כל שורה:
- להשמיע את הקובץ בעמודה `audio_path`
- לכתוב תמליל אמת בעברית בקובץ שמופיע בעמודה `reference_text_path`
3. להשאיר את שם הקובץ כמו שהוא (לפי `sample_id`).

## כללי כתיבה לתמליל אמת

1. לכתוב בדיוק מה שנאמר, בלי השלמות חופשיות.
2. בלי ניקוד.
3. אם לא ברור חלק מהמילה, לסמן `[לא ברור]`.
4. אין צורך בתגיות דובר בשלב הזה.
5. אם יש שקט/רעש בלבד - לרשום `[שקט]`.

## למה זה הכרחי

בלי תמליל אמת אי אפשר לחשב WER/CER אמיתי, ולכן אי אפשר לאמת בצורה מקצועית את איכות המודל הראשי.

## אחרי שתסיים

אני מריץ מיד:
1. בדיקת איכות מלאה למודל `ivrit-ai/whisper-large-v3`
2. דוח תוצאות מלא
3. עדכון KPI סופיים לאישור שלך
