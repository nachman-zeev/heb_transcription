# מדריך מהיר למשתמש (ללא פקודות יומיות)

המטרה: שהמערכת תעבוד בשבילך, עם לחיצה אחת להפעלה.

## 1) הפעלה בלחיצה אחת

1. הפעל את הקובץ:
`c:\tools\Projects\heb_transcription\scripts\local\start_local_easy.bat`
2. הדפדפן ייפתח אוטומטית על:
`http://127.0.0.1:8090`

## 2) כניסה ראשונה למערכת (מתוך ה-UI)

1. במסך הכניסה פתח:
`הרשמה ראשונית (Bootstrap, פעם אחת)`
2. מלא שם חברה, אימייל וסיסמה ולחץ:
`צור משתמש ראשוני`
3. התחבר עם אותם פרטים.

## 3) שימוש שוטף (מתוך ה-UI)

1. באזור `תמלול חדש` בחר קובץ אודיו (`.wav` או `.mp3`).
2. לחץ `העלה וצור Job`.
3. המתן לסטטוס `completed`.
4. הורד `TXT/SRT/DOCX` מתוך פרטי השיחה.

## 4) קבצי דוגמה לבדיקה

אפשר לבחור קבצים מתוך:
`c:\tools\Projects\heb_transcription\Recordings Examples`

## 5) עצירה בלחיצה אחת

הפעל:
`c:\tools\Projects\heb_transcription\scripts\local\stop_local_easy.bat`
